About Smart Sorting

AI-driven solution to accurately classify healthy vs rotten fruits and vegetables, helping reduce food waste.

Our Mission

To provide an efficient, AI-powered system that leverages transfer learning for detecting the freshness of fruits and vegetables, minimizing waste and supporting sustainable food supply.

Using a fine-tuned deep learning model based on VGG16 architecture, Smart Sorting delivers high-accuracy classification for 28 classes of produce — both healthy and rotten.

Our web application enables real-time quality assessment through an easy-to-use interface, empowering farmers, vendors, and consumers alike.

Project Timeline

Data Collection & Preparation

Compiled a comprehensive dataset with healthy and rotten fruit/vegetable images from Kaggle and other sources.

Model Training

Fine-tuned VGG16 architecture using transfer learning and augmentation techniques for high accuracy classification.

Web Application Development

Developed a Flask-based web app to upload images and receive instant classification results.

Testing & Deployment

Tested extensively on various produce samples and deployed the app for real-world use.
